Tetris Game Documentation

Controls

S: Start Game

ESC: End Game

Space: Rotate Shape

Left Arrow: Move Shape Left

Right Arrow: Move Shape Right

Down Arrow: Drop Shape Down

P: Pause Game

Game Rules

The game starts at level 1 and can reach up to level 10. With each new level, the difficulty increases - shapes start falling faster.

Scoring

Maximum accumulated score is saved after each game.
